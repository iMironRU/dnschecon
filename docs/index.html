Как работает
От изменения записи
до уведомления
1

Создай наблюдение

Через Telegram Mini App или закоммить YAML-файл в watches/. Укажи домен, тип записи и ожидаемое значение.

2

Durable Object запускает цикл опроса

Worker создаёт DO, который будит себя через alarm() с умным backoff: 30s → 30s → 60s → 2m → 5m → 10m → 30m → 1h.

3

Параллельный опрос всех резолверов

Каждый раунд — параллельные DoH JSON запросы к Google (с разными ECS-подсетями) и Cloudflare. Retry-логика при ошибках.

4

Прогресс в реальном времени

Telegram-сообщение обновляется на месте после каждого раунда. Mini App показывает кольцо прогресса и детали каждого резолвера.

5

Уведомление о сходимости

Когда все резолверы (или кворум) вернули ожидаемое значение N раундов подряд — ты получаешь финальное Telegram-уведомление.
